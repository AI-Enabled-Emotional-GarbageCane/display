# display

顯示與互動 UI — [AI 情緒垃圾筒](https://github.com/AI-Enabled-Emotional-GarbageCane/ai-enabled-emotional-garbagecane) 的顯示子系統。

## 職責

- 顯示螢幕互動 UI
- 訂閱 vision 推論結果並渲染
- 互動 Option 按鈕邏輯
- 簡報 / 影片 Demo / 海報素材

## 介面

- 接收 vision 的 `recognition_result` 並顯示
- 發送 `user_action` 指令給 firmware

詳細 API 契約見 [monorepo `docs/api-contract.md`](https://github.com/AI-Enabled-Emotional-GarbageCane/ai-enabled-emotional-garbagecane/blob/main/docs/api-contract.md)。
